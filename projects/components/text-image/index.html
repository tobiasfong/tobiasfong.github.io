Text + Image

A flexible section for editorial blocks, feature callouts, or project summaries.

Default layout

A clean two-column block that stacks on mobile.

Use this when you want a short story next to an image. It works well for project pages, capability pages, and landing sections.

  • Responsive: stacks on smaller screens
  • Consistent image window
  • Easy to reverse layout

See Projects Back to Components

Reverse layout

Add text-image--reverse to flip the order so the image appears on the left on desktop.

This is useful when you want visual rhythm down a page, alternating image placement by section.

Tighter variant

Add text-image--tight for a smaller gap and a slightly smaller image window.
